Redes en docker

En Docker, una red o "network" es un conjunto de contenedores Docker que se comunican entre sí, permitiendo que las aplicaciones distribuidas se ejecuten de manera eficiente y segura. Las redes en Docker facilitan la conexión y la comunicación entre contenedores, lo que es fundamental para crear aplicaciones complejas que constan de múltiples servicios.

Comandos de redes

  • ● Crear una red --> docker network (nombre_network)

  • ● Listar las redes que tenemos --> docker network ls

  • ● Ver la info de una red --> docker network inspect (nombre_network)

  • ● Eliminar una red --> docker network rm (nombre_network)

  • ● Eliminar todas las redes no utilizada --> docker network prune

Conectar/desconectar contenedores a redes

  • ● Conectar contenedor a una red --> docker network connect (nombre_network) (nombre_contenedor)

  • ● Desconectar contenedor a una red --> docker network disconnect (nombre_network) (nombre_contenedor)
