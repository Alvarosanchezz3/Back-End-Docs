Comandos más importantes

Gestión de imágenes

  • ● Descargar imagen de DockerHub --> docker pull (nombre de imagen)

  • ● Muestra las imágenes de Docker en local --> docker images

  • ● Elimina una imagen local--> docker rmi (nombre de imagen)

Gestión de contenedores

  • ● Crea y ejecuta un contenedor --> docker run --name (nombre_del_contenedor) (nombre_de_la_imagen)

  • ● Muestra solo los contenedores en ejecución --> docker ps

  • ● Muestra todos los contenedores --> docker ps -a

  • ● Detiene un contenedor en ejecución por su ID. --> docker ps -a

  • ● Elimina un contenedor por su ID --> docker rm (ID_del_contenedor)

  • ● Ejecuta un comando en un contenedor en ejecución --> docker exec -it (ID_del_contenedor) (comando)

  • ● Muestra los registros de un contenedor --> docker logs (ID_del_contenedor)

Gestión de volúmenes

  • ● Lista los volúmenes Docker disponibles --> docker volume ls

  • ● Crea un nuevo volumen --> docker volume create (nombre_volumen)

  • ● Muestra detalles sobre un volumen específico --> docker volume inspect (nombre_volumen)
