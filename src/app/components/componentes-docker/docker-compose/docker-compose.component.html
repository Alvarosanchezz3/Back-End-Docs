Docker Compose

Docker Compose es una herramienta que permite definir y gestionar aplicaciones multi-contenedor en Docker de forma sencilla. Permite definir la configuración de una aplicación, incluyendo los servicios, las redes, los volúmenes y otras configuraciones necesarias para ejecutarla, todo en un único archivo YAML llamado docker-compose.yml

Ejemplo Aplicación con base de datos

  • ● Servicio "db": Usa la imagen de MySQL, configura una base de datos y establece una contraseña para el usuario root.
  • ● Servicio "my-app": Construye una imagen desde el directorio actual (.), asigna el puerto interno de docker 9191 al puerto 8585 del host y especifica que dependa del servicio db.
                    
                         version: '3.1'
                        
 services:
                           db:
                             image: mysql
                             restart: always
                             environment: 
                               MYSQL_DATABASE: usersDB
                               MYSQL_ROOT_PASSWORD: root
                             container_name: mysql-msvc-users
                           my-app:
                             build:  .
                             image: msvc-users-img
                             ports: 
                               - "8585:9191"
                             container_name: msvc-users
                             depends_on: 
                               - db
