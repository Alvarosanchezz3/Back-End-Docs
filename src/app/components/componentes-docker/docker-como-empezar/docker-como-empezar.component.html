Docker

Docker es una plataforma de contenerización que permite ejecutar aplicaciones en entornos aislados llamados contenedores, estos incluyen todo lo necesario para que una aplicación funcione, ya sea en local, en servidores de desarrollo o en la nube.

Beneficios

  • ● Portabilidad: Empaqueta tus apps y sus dependencias y las ejecuta sin preocuparse de sistemas operativos.
  • ● Aislamiento: Las apps no afectarán ni interferirán con otras del mismo host.
  • ● Eficiencia: Los contenedores comparten el kernel del sistema operativo por lo que son más eficaces que las máquinas virtuales que usan un sistema operativo completo.
