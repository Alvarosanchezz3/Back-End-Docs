Servicio S3

Amazon S3 (Simple Storage Service) es el servicio de almacenamiento de objetos de Amazon Web Services de tipo PaaS (Platform as a Service). Es la solución más común para almacenar datos en la nube de una forma segura, eficiente y escalable. Con este servicio, podemos almacenar información de cualquier tipo, que se tratará como objetos. Por ejemplo, podemos usar S3 para almacenar fotografías, archivos de texto, código de webs estáticas, logs, backups, vídeos, etc.

1. Preparación para crear proyecto

  • Antes de empezar con el proyecto debemos ir a nuestra consola de AWS y realizar estos pasos:
  • 1) Crear bucket de S3: Buscamos el servicio S3 → buckets → Crear bucket. Elegimos la región más cercana a nosotros, le ponemos nombre al bucket (es único en el mundo) y le damos a "Crear bucket"
  • 2) Crear usuario de acceso: Buscamos el servicio IAM (Identity and Access Management) → Administración del acceso → Usuarios → Crear usuario. Elegimos “Adjuntar políticas directamente” y elegimos el permiso “AmazonS3FullAccess”.
  • 3) Conseguimos las claves de acceso de usuario: Nos metemos en el usuario que hemos creado y vamos a: Credenciales de seguridad → Crear clave de acceso → Aplicación fuera de AWS → Crear clave de acceso
